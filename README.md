# Simple Interest Calculator

## Description

This project is a simple interest calculator that calculates the interest amount using principal amount, rate of interest, and time period.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Inputs

- Principal Amount
- Rate of Interest
- Time Period

## Example

Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = (1000 × 5 × 2) / 100

Result = 100

## Features

- Calculates simple interest
- Easy to use
- Provides accurate results
